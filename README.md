### Instructions

1. Install dependencies

```bash
sudo apt update
sudo apt-install qt6-base-dev build-essential cmake qt6-svg-dev
```

2. Build the project:

```bash
mkdir build
cd build
cmake ..
make
```

3. Run the project:

```bash
./grandmaster
```
